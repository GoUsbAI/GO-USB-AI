# Core Commands

This page lists only the few commands most users need. For the full surface, see [Command Index](/en/guide/commands).

## Start and stop

```bash
go-usb-ai start
go-usb-ai restart
go-usb-ai stop
```

## Status and diagnostics

```bash
go-usb-ai status
go-usb-ai doctor
```

## Version and update

```bash
go-usb-ai --version
go-usb-ai update
```

## Autostart checks

```bash
go-usb-ai service autostart status
go-usb-ai service autostart doctor
```

## Remote access checks

```bash
go-usb-ai remote status
go-usb-ai remote doctor
```

## Usage principle

For first use, you only need `start`, `status`, `doctor`, and `stop`.  
Look up other commands only when the scenario requires them.
