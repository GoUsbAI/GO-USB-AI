# @go-usb-ai/ui

Official GoUsbAi Web UI package.

This package is consumed by `go-usb-ai` and also published for standalone integration/development use.

The reusable chat primitives now live in `@go-usb-ai/agent-chat-ui`. This package keeps the GoUsbAi host shell, presenter/store wiring, config pages, and product-specific application integration.

## Install

```bash
npm i @go-usb-ai/ui
```

## Development

```bash
pnpm -C packages/go-usb-ai-ui dev
pnpm -C packages/go-usb-ai-ui build
```
